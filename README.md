# Button 按钮

基于 <用户界面设计语言及交互标准> 扩展 antd 按钮组件，样式，confirm 弹窗，以及异步 loading 状态，默认间距等进行标准化。

## 何时使用

如在包含表格页面中，表格的操作按钮和操作列需要用到按钮的时候。

## 安装

```bash
# pnpm
pnpm add @linewell/button -E

# yarn
yarn add @linewell/button -E

# npm
npm install @linewell/button -E
```
